Parking Automation System
Parking Automation System is a Python-based computer vision project designed to monitor parking slots, detect occupied and empty spaces, track reserved parking slots, and identify parking violations using video and image processing. The system uses OpenCV to analyze parking area footage, compare live frames with reference images, and display the parking status visually with colored overlays.
The project allows parking slots and violation zones to be marked manually using a separate drawing tool. These marked areas are saved and later used by the main program to monitor each parking space. Free slots are shown in green, occupied slots are shown in blue, reserved slots are shown in pink, and violation areas are highlighted when a vehicle stays in a restricted zone for more than the allowed time.
The system also supports reservation-based parking management. It reads reservation data from a JSON file, checks which slots are currently reserved, and updates the reserved slot list automatically. When a reserved slot becomes occupied, the system captures a snapshot of that slot and saves it for number plate verification.
This project is connected with a number plate recognition module. The parking system starts the number plate detection script in the background, captures reserved slot images, extracts vehicle plate numbers, and compares the detected plate with the registered plate from the reservation data. If the detected plate does not match the registered plate, the system triggers an alert.
The project also includes parking violation detection. If a vehicle is detected inside a marked violation zone for a fixed duration, the system saves a violation image and triggers an external alert script. This helps identify unauthorized parking or vehicles parked in restricted areas.
The main technologies used in this project are Python, OpenCV, NumPy, Pickle, JSON, threading, SimpleAudio, and text-to-speech support. OpenCV is used for video processing, slot detection, polygon drawing, and live visual display. JSON files are used to manage reservation and parking status data.
To run the project, install the required Python packages such as opencv-python, numpy, simpleaudio, and pyttsx3. After installing the dependencies, first use draw.py to mark parking slots and violation zones on the parking layout images. Then run main.py to start the live parking monitoring system.
The main file of the project is main.py, which handles live video processing, parking slot detection, reserved slot tracking, violation detection, popup display, and real-time parking status updates. The reservation.py file manages reserved slot snapshots and compares detected number plates with registered reservation details. The draw.py file is used to manually draw and save parking and violation areas.
Before running the project on another system, update the file paths inside the Python files according to your local folder structure. Some paths currently point to local directories for reservation data, number plate detection, violation images, and parked slot JSON files.
This project can be improved further by adding a web dashboard, replacing hard-coded paths with a configuration file, improving vehicle detection with deep learning models, adding database support for reservations, and creating a cleaner user interface for parking administrators.
